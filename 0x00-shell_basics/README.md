File descriptions

0-current_working_directory: Prints full path of your current working directory

1-listit: Lists all files of your current working directory

2-bring_me_home: Returns to home directory

3-listfiles: Lists all files in current working directory in long format

4-listmorefiles: List all files current working directory in long format, including hidden files

5-listfilesdigitonly: List all files in current working directory in long format, including hidden files, with user and group IDs displayed numerically

6-firstdirectory: Creates a directory called 'my_first_directory' in /tmp folder

7-movethatfile: Moves a file called betty from the /tmp folder to /tmp/my_first_directory

8-firstdelete: Deletes file called betty from /tmp/my_first_project

9-firstdirdelete: Deletes directory called my_first_directory from /tmp

10-back: Changes the current working directory to the previous one
