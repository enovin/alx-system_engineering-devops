File Descriptions

0-alias: Create an alias "ls" with a value rm *

1-hello_you: Print "hello user" where user is the current Linux user

2-path: Add /action to the path, making it the last directory the shell looks into when looking for a program

3-paths: Display the number of directories in PATH

4-global_variables: List environment variables

5-local_variables: List all local variables and environment variables, and functions

6-create_local_variable: Create local variable BEST with value School

7-create_global_variable: Create global variable BEST with value School

8-true_knowledge: Print the result of the addition of 128 with the value stored in TRUEKNOWLEDGE
