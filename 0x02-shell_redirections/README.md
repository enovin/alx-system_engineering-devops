File Descriptions

0-hello_world: Print 'Hello World!', followed by anew line to the final output

1-confused_smiley: Display a confused smiley

2-hellofile: Display the contents of the /etc/passwd file

3-twofiles: Display the contents of the /etc/passwd and /etc/hosts files

4-lastlines: Display the last ten lines of the /etc/passwd file

5-firstlines: Display the first ten lines of the /etc/passwd file

6-third_line: Display the third line of the file "iacta"

7-file: Create a file named \*\\'"Best School"\'\\*$\?\*\*\*\*\*:) with content "Best School"

8-cwd_state: Output the result of ls -la into the file ls_cwd_content and overwrittes the file if it already exists

9-duplicate_last_line: Duplicate the last line of the file "iacta"

10-no_more_js: Delete all regular files (not directories) with a .js extension that are present in the working directory and all of its subfolders

11-directories: Count the number of directories and subdirectories (including hidden directories) in the current working directory, not taking into account the current and parent directories

12-newest_files: Display the first 10 newest files in the current directory

13-unique: Take a list of words as input and prints words that only appear exactly once, in a sorted order

14-findthatword: Display lines containing the pattern "root" from the file /etc/passwd

15-countthatword: Display the number of lines that contain the word "bin" in the file /etc/passwd

16-whatsnext: Display lines containing the pattern "root" and 3 lines after them in the file /etc/passwd
